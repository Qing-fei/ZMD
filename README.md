# ZMD 自动化工具

一个基于 PyQt5 + OpenCV + OCR 的游戏自动化工具。

---

## 功能

- 仓库搬运（支持武陵 ↔ 谷地）
- 自动精锻
- 委托自动刷新与领取（OCR识别佣金）
- 点位管理 + 动作安排（可扩展）（测试功能未完善，酌情使用）

---

## 使用方法

### 1. 下载

在 Release 页面下载压缩包并解压。

### 2. 运行

## 注意事项
确保解压后的文件路径没有中文
包含2个文件夹和一个exe文件，请将他们置于同一文件夹内
杀毒软件可能误报

管理员运行（不是游戏里那个管理员，请鼠标右键点击exe文件→以管理员身份运行）：
main.exe

### 3.使用说明
i.打开软件后，请先绑定游戏窗口，如果你的窗口名不是Endfield，请在窗口绑定界面自行输入
ii.关于仓库搬运：参数waittime是点击切换武陵/谷地后等待3s然后按esc退出，如有需要可自习调试
    可输入背包容量和搬运总数计算循环次数
  关于武陵委托：最高数量是在识别失误后的校验措施，可自行根据游戏版本调整
  关于动作安排和窗口绑定：测试功能，酌情使用

---

## 技术栈

- PyQt5（界面）
- OpenCV（图像处理）
- pytesseract（OCR）
- PyInstaller（打包）
